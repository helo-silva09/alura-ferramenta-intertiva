Por que Java script é Importante?
É uma das três tecnologias essenciais da World Wide Web
(junto com HTML e CSS). Sem JavaScript, a maioria dos sites modernos seria estática e sem vida.
É uma habilidade fundamental para qualquer desenvolvedor web.
