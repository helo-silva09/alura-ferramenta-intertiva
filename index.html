Por que Java script é Importante?
É uma das três tecnologias essenciais da World Wide Web
(junto com HTML e CSS). Sem JavaScript, a maioria dos sites modernos seria estática e sem vida.
É uma habilidade fundamental para qualquer desenvolvedor web.
oque faz o Java script no celular?
O JavaScript, em relação aos celulares, não é um aplicativo que você "instala" ou "localiza" como um programa autônomo.
Ele é uma linguagem de programação que está integrada aos navegadores
(como o Chrome, Safari, etc.) para permitir a execução de scripts
e funcionalidades interativas nas páginas web que você acessa.
